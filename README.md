Weather App 

A simple and clean Android weather application that shows real-time weather information using the OpenWeather API.

Features
- Search weather by city
- Real-time temperature and weather conditions
- User-friendly and minimal UI
- Fast and accurate API results

Tech Used
- Kotlin
- Android Studio
- OpenWeather API
- XML UI Design

How to Run
1. Clone the repository
2. Open in Android Studio
3. Add your OpenWeather API key in the code
4. Run the app on an emulator or device
